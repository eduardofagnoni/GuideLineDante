Estamos utilizando o Bootstrap como base para o sistema de grids. Para maiores informações sobre a aplicação das grids consulte a documentação do bootstrap disponível em: getbootstrap.com

Todo conteúdo das páginas devem estar dentro da classe "container"

		
<div class="container">	... </div>
